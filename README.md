# Android Encrypt Library
android library of encrypt utils

### Installation Adding to project
1.Add the Jcenter repository to your build file
```
buildscript {
    repositories {
        jcenter()
    }
}
```
2.Add the dependency
```
dependencies {
    implementation 'gapp.season:EncryptLib:0.0.1'
}
```
